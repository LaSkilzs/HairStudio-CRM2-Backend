## Shear Elegance

This is the api for the shear elegance clientside app. The api features

1. Authorization/Authentication.
2. Relational Database Design(normalization, associations).
3. Data Serializing
4. Testing
5. API versioning

## Technologies

1.  React
2.  Rails Api
